# JARVIS OS - Advanced AI Operating System

## Installation & Usage

### Prerequisites
- Python 3.10+
- Ollama (for local AI)
- Windows 10/11

### Quick Start
```bash
# Install dependencies
pip install -r requirements.txt

# Run JARVIS OS
python src/main.py
```

## Features
- 🤖 AI Assistant with Ollama (offline, no API keys)
- 🎨 8K/16K Image Generation (Stable Diffusion)
- 🎬 Video/Movie Generation (Stable Video Diffusion)
- 💻 Coding Agent (50+ languages)
- 📚 Learning Engine (GitHub, web, books)
- 🏗️ Project Builder (React, Electron, Django, etc.)
- 🗣️ Multi-language Voice I/O (8 languages)
- 💾 50GB Storage Cache
- ⚙️ OS Control with autostart
- 🖥️ System tray support

## Tech Stack
- PyQt6 - Premium UI
- PyTorch - AI/ML
- Diffusers - Image/Video Generation
- Whisper - Voice recognition
- Ollama - Local LLM

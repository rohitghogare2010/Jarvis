## 🚀 Overview

**Jarvis** is an advanced AI-powered desktop assistant designed exclusively for Windows. It combines a stunning 3D holographic glassmorphism UI with powerful AI capabilities to provide an unparalleled computing experience. Jarvis learns continuously, generates images and videos, writes code, fixes bugs automatically, and controls your entire operating system through natural voice or text commands.

## 🛠️ Installation

### Prerequisites

```bash
# Install Python 3.10+
# Install Node.js 18+
# Install Ollama (for local AI)
# Install Git
```

### Quick Install

```bash
# Clone the repository
git clone https://github.com/rohitghogare2010/Jarvis.git
cd Jarvis

# Run setup
python setup.py install

# Launch Jarvis
python Jarvis.py
```

## 🏃 Quick Start

1. **First Launch**: Jarvis will initialize AI models and create necessary directories
2. **Voice Setup**: Speak "Hey Jarvis" to activate (configurable)
3. **Text Mode**: Type commands directly in the chat interface
4. **System Control**: Use natural language to control Windows

## 🔧 Configuration

Edit `config.json` to customize:

```json
{
  "jarvis_name": "Jarvis",
  "activation_phrase": "Hey Jarvis",
  "voice": {
    "language": "en-US",
    "rate": 1.0,
    "volume": 1.0
  },
  "ai": {
    "model": "llama3",
    "provider": "ollama"
  },
  "storage": {
    "path": "./storage",
    "encrypted": true
  }
}
```
